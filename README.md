## ScrewDriver Project

The ScrewDriver Project is an interactive racing game built in Unreal Engine where players race using screwdriver-themed vehicles. The project focuses on creativity, gameplay mechanics, and environmental interaction, offering a lighthearted and engaging racing experience.

## Project Overview
ScrewDriver is an immersive racing game, featuring nuts, bolts, and a screwdriver as your car! You will race through the Toolbox obstacle course against an opposing screwdriver. If you can catch up to yor oponent, you can use your magical ability of destruction to unscrew the opposing driver, ultimately dismanteling their vehicle. This project was created using Unreal Engine.

## Project Purpose
The ScrewDriver Project was created to explore game development principles, learn Unreal Engine workflows, and experiment with unique, creative vehicle designs. The project highlights teamwork, problem-solving through game mechanics, and skill-building in Blueprint scripting.
